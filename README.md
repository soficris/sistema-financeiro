# Sistema Financeiro 
Este projeto consiste no desenvolvimento de uma aplicação em PHP para gerenciar solicitações de clientes em uma empresa de serviços financeiros.

Atualmente, as solicitações (como ajuste de limite de crédito, atualização de dados e pedidos de novos serviços) são controladas por e-mails e planilhas, o que gera atrasos, falta de organização e dificuldade em acompanhar responsabilidades e prazos.

O sistema foi proposto para centralizar e organizar essas demandas, permitindo registrar, acompanhar e atualizar solicitações de forma mais eficiente, garantindo maior controle interno e transparência no atendimento aos clientes.
---
## Funcionalidades 
👤Clientes
Armazenamento das seguintes informações:

- ID
- Nome completo
- CPF
- E-mail
- Telefone

📋 Solicitações
Cada solicitação contém:

- ID da solicitação
- ID do cliente
- Descrição do serviço solicitado
- Urgência: baixa, média ou alta
- Status: pendente, em andamento ou finalizada (padrão: pendente)
- Data de abertura
- ID do funcionário responsável (opcional no cadastro inicial)
